# Sokoban
Sokoban arcade game engine
Graphics by Sebastian 'xa' Rosik

# Mapping
The map template working with my engine looks like this:
                
                
                
   wwwwww       
  ww..x.w       
  w.s.w.w       
  w.xs..w       
  w..wsww       
  wx.p.w        
  wwwwww        
                
                
                
                
                
                
                
16 rows and 16 line.
You must draw map design by letter.
space is an empty field,
"c" is normal wall,
"." is normal floor,
"w" is castle wall,
"p" is player,
"s" is stone,
"x" is stone floor,
